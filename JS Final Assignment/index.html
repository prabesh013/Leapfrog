How To Play

1. Choose a level based on your arithmetics knowledge. (Easy, Medium, Hard)

2. An expression based on the chosen level will appear. You need to draw in the canvas the corresponding digit answer.

3. A timer is set. You need to draw within the specified time. If failed to draw in the given time, game is over.

4. Score is based on the complexity of the expression.

  • Easy: 1 (20 sec)
  • Medium: 2 (25 sec)
  • Hard: 3 (35 sec)

Caution: Donot touch the canvas border while drawing. If touched, canvas will be cleared automatically and you need to redraw.
